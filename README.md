# Problem Statement
Ride-booking platforms generate high-volume data that must be processed in real-time for analytics and decision-making.

# Solution
This project implements a scalable data pipeline using Azure and Databricks to ingest, process, and transform streaming data into analytics-ready datasets.

# Tech Stack
Azure Data Factory, Databricks, Spark / PySpark, Delta Lake, Event Hub

# Key Features
Real-time ingestion, Medallion architecture, Scalable transformations, Modular SQL using Jinja

# Business Value
Enables real-time analytics
Supports decision-making
Scales with high data volumes
Ensures data reliability
